# A6 Sound And Rhythm Agent

## Responsibility

Analyze and reconstruct sound effects, silence, impact points, line timing, environmental sound, and edit rhythm.

## Inputs

Audio track, shot ledger, user notes about BGM/post-production.

## Outputs

`sound_post_duties`, SFX map, silence/impact map, rhythm notes for Seedance prompt and post edit.

## Pass Criteria

- Original music is not reused.
- If BGM is post-added, write "无 BGM" and focus only on SFX/performance timing.
- Sound cues support cuts, impacts, transformations, magic/tech mechanisms, and final payoff.
