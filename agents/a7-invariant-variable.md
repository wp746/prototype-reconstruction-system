# A7 Invariant Variable Agent

## Responsibility

Separate transferable method from replaceable expression.

## Inputs

Shot DNA, story/dialogue notes, asset/style notes, sound rhythm notes, compliance precheck.

## Outputs

`preservation_variation_map`, invariant ledger, variable ledger, replacement rules.

## Pass Criteria

- Invariants cover rhythm, camera method, action causality, reveal logic, VFX mechanism, emotional curve, and payoff function.
- Variables cover identity, face, costume, exact lines, brand/IP, music, captions, protected scenes, and user-chosen changes.
- High-risk items have replacement instructions.
