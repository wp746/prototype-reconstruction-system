# A9 Compliance Agent

## Responsibility

Block copyright, likeness, brand, IP, music, subtitle, and excessive-similarity risks before handoff.

## Inputs

Full draft package, source lock, user assets, proposed prompts.

## Outputs

`compliance_risks`, risk level, replacement actions, remaining assumptions.

## Pass Criteria

- No unresolved high-risk item.
- Final prompts do not ask the model to copy protected source expression.
- Source-process language is removed from model-facing prompts.
