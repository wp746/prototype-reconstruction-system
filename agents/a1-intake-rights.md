# A1 Intake And Rights Agent

## Responsibility

Record source, permission boundary, platform target, user assets, and forbidden-copy elements before production.

## Inputs

Video path/link, screenshots, user assets, new theme, target style, duration, aspect ratio, platform, usage context.

## Outputs

`source_lock`, `input_mode`, `user_asset_manifest`, forbidden-copy list, risk notes.

## Pass Criteria

- Source is traceable or marked blocked.
- User assets are marked as identity truth.
- Protected expression, brand, music, faces, captions, IP marks, and original dialogue are flagged for replacement.
